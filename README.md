# 🖥️ Guia de Montagem de PC (TCC)

Um guia interativo desenvolvido para auxiliar usuários na escolha de componentes de hardware, personalizados de acordo com a sua necessidade específica: **Trabalho**, **Jogos** ou **Estudos**.

## 🚀 Sobre o Projeto
O objetivo deste site é simplificar a montagem de computadores, oferecendo curadorias de peças recomendadas para diferentes perfis de uso. Além das sugestões de builds, o projeto atua como uma central educativa, explicando a função de cada componente (CPU, GPU, RAM, etc.) e sua importância para o desempenho final.

## 🛠️ Funcionalidades
* **Navegação por Perfil:** Abas específicas para Jogos, Trabalho e Estudos.
* **Curadoria de Hardware:** Listagem de peças compatíveis e recomendadas para cada categoria.
* **Dicionário Técnico:** Explicações didáticas sobre o que cada peça faz e para que serve.
* **Design Responsivo:** Interface limpa focada na experiência do usuário (UX).

## 🧰 Tecnologias Utilizadas
* **HTML5:** Estruturação semântica do conteúdo.
* **CSS3:** Estilização moderna e layout responsivo.
* **Git/GitHub:** Controle de versão e hospedagem.

## 📁 Organização do Código
O projeto foi estruturado seguindo boas práticas de organização, separando as folhas de estilo por contexto:
- `jogos.css`, `trabalho.css`, `essendou.css` (estilos específicos).
- `index.html` (página principal).
